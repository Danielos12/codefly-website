Code Invaders

Zasiądź za sterami nowoczesnego statku kosmicznego i ochroń ziemię przed najeźdźcami z kosmosu w tym niebywałym połączeniu symulatora lotu oraz gry akcji, poznając przy tym rozbudowaną historię traktująca o wartości ludzkiego życia w maszynie wojennej. Dzięki rozbudowanemu systemowi sterowania którego pozazdrościłaby nie jedna gra symulacyjna a jednocześnie niezwykle przystępnego nawet dla niedzielnego gracza, doświadczysz ekscytujących walk kosmicznych jakie do tej pory widziałeś tylko w filmach.
